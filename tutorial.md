# RHex Video Tutorial Guide

Use this step-by-step guide to record your tutorial video. It covers the core features, including the newly added input enhancements.

## Phase 1: The Basics (Adding Colors)

**Action**: Start with a clean/empty palette (or just a few colors).
**Focus**: Sidebar Input.

1.  **Standard Input**: Click the hex input field. Type a standard hex code (e.g., `#FF5733`). Press **Enter** to add it.
2.  **Flexible Input (New)**: Clear the input. Type a hex code *without* the hash (e.g., `4287f5`). point out standard "Blue". Press **Enter**.
    *   *Highlight*: Notice it automatically formats to `#4287F5` and adds it to the grid immediately.
3.  **Random Color**: Click the "Dice" icon a few times to quickly populate the grid.

## Phase 2: Palette Management

**Action**: Interact with the main grid.

4.  **Reorder**: Click and drag a color to a new position.
5.  **Edit**: Tap a color to open the color picker. Change it slightly and Update.
6.  **Grid Size**: Go to the sidebar "Grid Size" slider. Slide it back and forth to show the grid resizing responsiveness.
7.  **Right-Click Actions**: Right-click (or context click) a color. Select "Generate Shades".
    *   *Showcase*: Add a few tint/shade variations from the dialog.

## Phase 3: Advanced Tools

**Action**: Use the tools at the bottom of the sidebar/menu.

8.  **Batch Add**: Click "Batch Add". Paste a list of colors (e.g., `red`, `00ff00`, `#0000ff`).
9.  **Image Import**: Click the Import icon (or shortcut). Select an image. Show the color extraction wizard. Pick a few colors and "Add to Palette".

## Phase 4: Output

**Action**: Finalize and Export.

10. **Theme Toggle**: Toggle Dark/Light mode to show UI adaptability.
11. **Export**: Click the Export button.
    *   Show the options: **CSS**, **Tailwind**, **PNG**, **Text**.
    *   Select **PNG** and save the image to demonstrate the visual export.

## Phase 5: Outro

**Action**: Show the final beautiful palette.
**Voiceover**: "That's how you create stunning palettes with RHex."
